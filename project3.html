Scripting

Beschrijving

Voor het vak scripting kregen we de examen opdracht om een Powershell-script te schrijven die een windows server 2025 volledig kan instellen. Hiervoor maakten we ook een interactief keuzemenu dat je zelf kan bepalen welk je wilt instellen of je het volledig automatisch wou laten uitvoeren. Ook moesten we ervoor kunnen zorgen dat het werkt op een windows 11 client om deze toe te voegen aan het domein. Voor al de info wat hij exact moet instellen leest het script dit uit bestanden die we van de leerkracht kregen zoals onder andere csv en xml bestanden en mochten we daar niets in aanpassen. Hierin waren ook bewust fouten ingezet waar wij voor moesten zorgen dat het script aan failover kan doen en hier niet op vast loopt maar ervoor zorgt dat hij de data toch kan meenemen.

Wat kan het script:

  • Netwerkkaarten instellen
  • Basisconfiguratie
  • AD instellen
  • OU's aanmaken
  • Users aanmaken
  • Mappen aanmaken
  • Shares aanmaken
  • Rechten toekennen
  • Volledig automatische installatie
  • Windows 11 client instellen en toevoegen
