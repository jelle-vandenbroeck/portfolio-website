Microsoft Defender Application Control (MDAC/WDAC)

Microsoft Defender Application Control is een ingebouwde beveiligingsmaatregel die standaard in elke moderne windows operating system zit. Het zorgt er namens voor dat je als System admin rechten kan instellen per device of per user om bepaalde applicaties te kunnen blokkeren. Zo wil je bijvoorbeeld dat eindgebruikers geen Powershell of Command Prompt mogen openen en gebruiken, dit kan je dan heel gemakkelijk gaan instellen in bijvoorbeeld GPO's als je met Active Directory werkt of op de pc zelf in AppLocker of als je in de cloud werkt kan het ook via Microsoft Intune.

Muraena

Muraena is een moderne bijna transparante reverse proxy phishing toolkit die gemaakt is om phishing campagnes zoveel mogelijk te kunnen automatiseren. Het steelt de sessie-tokens van het slachtoffer en kan met de samenwerking van Necrobrowser de sessie overnemen. Als het slachtoffer op de phishing link klikt word hij/zei omgezeild naar het phishing-domein die er exact hetzelfde uitziet als de echte inlogpagina en word nadien wel naar het juiste domein geleid zodat die er niets van merkt omdat alle gegevens door Muraena worden doorgestuurd naar het echte domein en het slachtoffer dus daar aanmeld. Door Muraena te laten samenwerken met Necrobrowser worden de gegevens naar Necrobrowser gestuurd en gaat die met de gestolen tokens aanmelden op de website en kan bijvoorbeeld de onedrive volledig downloaden met alle documenten die daarop staan of een screenshot maken als bewijs voor de phishing campagne.
